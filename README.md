# Traffic Vehicle Detection and Violation Alert System

## Overview

A real-time traffic monitoring system that detects and records traffic violations using Computer Vision and Machine Learning.

## Features

- Vehicle Detection
- Vehicle Tracking
- Signal Jump Detection
- Wrong Way Detection
- Overspeed Detection
- Number Plate Recognition
- Violation Evidence Capture
- Email/SMS Alerts

## Tech Stack

- Python
- OpenCV
- YOLO
- SQLite
- Flask

## Future Enhancements

- Helmet Detection
- Triple Riding Detection
